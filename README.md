SMS Spam Classifier 
A Machine Learning powered SMS filter that distinguishes between Ham (legitimate) and Spam messages using Natural Language Processing (NLP).

This project addresses the "Class Imbalance" problem in text classification. 
By analyzing a dataset of over 5,000 SMS messages, I built a pipeline that cleans raw text, extracts statistical features,
and uses a Naive Bayes model to identify spam with high precision.

Key Insights:
Spam messages are significantly longer on average (approx 138 characters) compared to Ham messages (approx 70 characters).
The dataset is imbalanced, with roughly 86.6% Ham and 13.4% Spam.
